Detta är en laboration i _Webbutveckingsprogrammet_ på Mittuniveristetet.

[Min webbplats]()

## Syfte
Syftet med denna laboration är att ge dig praktisk erfarenhet av att skapa en "att göra"-applikation med TypeScript och använda principer för objektorienterad programmering (OOP). Du kommer även att bekanta dig med användningen av interfaces, olika typer och lokal lagring (LocalStorage).

## Uppgift
Du ska skapa en "att göra"-applikation där användaren kan lägga till nya uppgifter (todos), markera dem som klara och visa en lista över sina todos. Dessutom kommer du att använda olika typer för att hantera prioritering av uppgifterna och lagra listan med hjälp av LocalStorage.

## Lösning
Här kan du läsa om vilka funktioner som finns, vad som används samt hur repot kan installeras lokalt i din dator.

### Funktioner
* **Lägga till:** Uppgifter kan läggas till via formulär, beskrivning och prioritet (1-3) anges.
* **Markera uppgift klar:** Inlagda uppgifter kan markeras som klara då stryks texten över och knappen ändrat utseende.
* **Ta bort uppgift:** Uppgifter kan tas bort från listan genom att trycka på knappen ta bort.
* **LocalStorage:** Lagring av uppgifterna sker automatiskt med LocalStorage och sparas mellan sessioner.
* **Validering:** Inputfälten valideras, fälten får ej vara tomma och giltigt nummer för prioritet 1, 2 eller 3. Felmeddelande visas vid ej giltig input.

### Har används för uppgiften
* HTML
* CSS
* TypeScript
* Vite
* GitHub Actions

### Installation
1. Klona repot
    ```sh
    git clone https://github.com/safe-pizza/DT208G_Laboration2.git
    ```

2. Gå in i projektmappen
    ```sh
    cd DT208G_Laboration2
    ```

3. Installera
    ```sh
    npm install
    ```

4. Starta en utvecklingsserver
    ```sh
    npm run dev
    ```

Nu finns projektet tillgängligt för dig att köra lokal på din dator via localhost.
